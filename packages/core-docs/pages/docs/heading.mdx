# Heading
<Heading.H1 type="slab" size="l" css={css`grid-column: 1 / span 12;`}> Type Slab Size L </Heading.H1>
<Heading.H2 type="slab" size="m" css={css`grid-column: 1 / span 12;`}> Type Slab Size M </Heading.H2>
<Heading.H3 type="slab" size="s" css={css`grid-column: 1 / span 12;`}> Type Slab Size S </Heading.H3>
<Heading.H4 type="regular" size="s" css={css`grid-column: 1 / span 12;`}> Type Regular Size S </Heading.H4>
<Heading.H5 type="regular" size="m" css={css`grid-column: 1 / span 12;`}> Type Regular Size M </Heading.H5>
<Heading.H6 type="regular" size="l" css={css`grid-column: 1 / span 12;`}> Type Regular Size L </Heading.H6>

Компонент заголовка. Используется для отображения заголовков. Имеет 6 различных уровней.
Заголовки разных уровней визуально никак не отличаются друг от друга, но они важны для семантики. 
Может быть только один заголовок первого уровня.

---------------------------------------
## Свойства
Имя  | Допустимые значения | Значение по умолчанию | Комментарий
---- | ------------------- | --------------------- | ----------------
type | slab \| regular     |                       | Стиль начертания
size | s \| m \| l         |                       | Размер заголовка

---------------------------------------
## Код
```
  import { Text } from '@asmy/core-design'

  <Heading.H1 type='slab' size='l'>
    Заголовок первого уровня
  </Heading.H1>
```
