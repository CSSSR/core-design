Компонент списка. Используется для создания ненумированных списков чего-либо. 
Не имеет каких-то специальных свойств. Каждый пункт это компонент `ListItem`.

---------------------------------------
## Код
```
  import { List, ListItem } from '@csssr/core-design'

  <List>
    <ListItem>проснуться</ListItem>
    <ListItem>поработать</ListItem>
    <ListItem>заснуть</ListItem>
  </List>
```
