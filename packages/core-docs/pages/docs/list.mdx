# List

<List>
  <ListItem>проснуться</ListItem>
  <ListItem>поработать</ListItem>
  <ListItem>заснуть</ListItem>
</List>

Компонент списка. Используется для создания ненумированных списков чего-либо. Каждый пункт это компонент `ListItem`.

---

## Свойства

| Имя  | Допустимые значения | Значение по умолчанию | Комментарий                                                                                                                                            |
| ---- | ------------------- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| size | s \| m              | s                     | Определяет размер шрифта для дочерних компонентов `ListItem`. В `ListItem` используется компонент <a href='/docs/text'>`Text`</a> начертания `regular` |

---

## Код

```
  import { List, ListItem } from '@csssr/core-design'

  <List>
    <ListItem>проснуться</ListItem>
    <ListItem>поработать</ListItem>
    <ListItem>заснуть</ListItem>
  </List>
```
