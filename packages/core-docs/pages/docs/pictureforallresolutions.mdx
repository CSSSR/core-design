Компонент изображения. Используется для отображения адаптивных картинок, которые могут иметь различные размеры на разных брейкпоинтах.
Стоит использовать только для <a href='https://htmlacademy.ru/blog/useful/image/content-or-decor-img'>контентных изображений</a>. Для
неконтентных изображений используйте CSS свойство <a href='https://developer.mozilla.org/ru/docs/Web/CSS/background-image'>`background-image`</a>.
Брейкпоинты компонент изображения берет из темы.

Изображения, которые планируется использовать с этим компонентом, требуется подготовить:
* создать папки с названием используемых брейкпоинтов;
* в каждой папке должно содержаться 6 копий данного изображения: 3 png, 3wepb;
* все изображения должны быть оптимизированные под экраны с разной плотностью пикселей и содержать один из постфиксов: @1x, @2x, @3x;
* все изображения должны быть оптимизированны по характеристики качество изображения/вес изображения.

В результате у вас должна получиться примерно вот такая структура папок и файлов:

---------------------------------------
## Свойства компонента PictureForAllResolutions
Имя               | Допустимые значения | Значение по умолчанию                                  | Комментарий
----------------- | ------------------- | ------------------------------------------------------ | ----------------
image             | {}                  |                                                        | Объект, в котором описываются свойства передаваемого изображения.
customResolutions | string[]            | ['mobile.all', 'tablet.all', 'desktop.s', 'desktop.m'] | Брейкпоинты, которые будет использовать компонент. Их названия берутся из темы.

---------------------------------------
## Свойства пропса image
Имя                | Допустимые значения | Значение по умолчанию | Комментарий
------------------ | ------------------- | --------------------- | ----------------
pathToImagesFolder | string              |                       | Путь до папки где лежат изображения, указанный с помощью `require.context`
imageName          | string              |                       | Название изображения
extension?         | string              | png                   | Расширение изображения. Если расширений файлов не больше двух, то по тому расширению, которое не является `webp`, если больше двух, то нужно указать расширение, какое вы хотите что бы было дефолтным. Необязательный параметр. Компонент требует наличия `webp` разрешения для каждого изображения.
alt                | string              |                       | Описание изображения, используется поисковиками и читалками экрана.

---------------------------------------
## Код
```js
  import { PictureForAllResolutions } from '@csssr/core-design'

    image = {{

    }}
  />
```
