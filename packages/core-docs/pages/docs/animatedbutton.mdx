Анимированная кнопка. Используется в формах для отображения статуса запроса сохранениях данных на сервер.

---------------------------------------
## Свойства
Имя       | Допустимые значения  | Значение по умолчанию | Комментарий
--------- | -------------------- | --------------------- | ----------------
status    | pending \| submitting \| success \| fail     | Статус запроса

---------------------------------------
## Хендлеры
Имя       | Аргументы | Комментарий
--------- | --------- | ---------------
onClick   | event     | Функция, которая будет исполнятся при клике на кнопку

---------------------------------------
## Код
```
  import { AnimatedButton } from '@csssr/core-design'

  <AnimatedButton
    status="success"
    onClick={handleClick}
  >
    Кнопка
  </AnimatedButton>
```
