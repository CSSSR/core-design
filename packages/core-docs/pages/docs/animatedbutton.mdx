Анимированная кнопка. Используется в формах для отображения статуса запроса сохранениях данных на сервер.


---------------------------------------
## Свойства
Имя       | Допустимые значения                          | Значение по умолчанию | Комментарий
--------- | -------------------------------------------- | --------------------- | ----------------
kind      | primary \| secondary                         | primary               | Тема кнопки
status    | pending \| submitting \| success \| fail     | pending               | Статус запроса


---------------------------------------
## Хендлеры
Имя       | Аргументы | Комментарий
--------- | --------- | ---------------
onClick   | event     | Функция, которая будет исполняться при клике на кнопку

---------------------------------------
## Код
```js
  import { AnimatedButton } from '@csssr/core-design'

  <AnimatedButton
    status="success"
    onClick={handleClick}
  >
    Кнопка
  </AnimatedButton>
```
