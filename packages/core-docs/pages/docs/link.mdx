# Heading

Компонент ссылок. Используется для переходов на другие страницы. 

---------------------------------------
## Свойства
Имя      | Допустимые значения        | Значение по умолчанию | Комментарий
-------- | -------------------------- | --------------------- | ----------------
type     | list \| top_menu \| burger |                       | Стиль начертания
size     | s \| m \| l                |                       | Размер заголовка
href     | string                     |                       | Путь на который должна вести ссылка
external | true \| false              | false                 | Отвечает за то будет открываться ссылка в новом окне или в существующем

---------------------------------------
## Код
```
  import { Link } from '@asmy/core-design'

  <Link type="slab" size="l"> Link </Link>
```
