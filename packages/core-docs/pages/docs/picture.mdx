Компонент изображения. Используется для отображения картинок. Стоит использовать только для 
<a href='https://htmlacademy.ru/blog/useful/image/content-or-decor-img'>контентных изображений</a>.
Для неконтентных изображений используйте CSS свойство 
<a href='https://developer.mozilla.org/ru/docs/Web/CSS/background-image'>`background-image`</a>.

---------------------------------------
## Свойства
Имя | Допустимые значения | Значение по умолчанию | Комментарий
--- | ------------------- | --------------------- | ----------------
src | string              |                       | Ссылка на изображение
alt | string              |                       | Название изображения, используется поисковиками и читалками экрана

---------------------------------------
## Код
```
  import { Picture } from '@csssr/core-design'

  <Picture
    src='https://nosidebar.com/images/benefits-of-minimalism@2x.jpg'
    alt='Фотография помещения в стили минимализма'
  />
```
